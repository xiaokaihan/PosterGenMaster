# Assets 目录说明

此目录用于存放海报生成所需的资源文件。

## 必需文件

### template.jpg
- **说明**: 海报背景底图
- **格式**: JPG/JPEG 图片
- **建议尺寸**: 根据实际需求设置（例如：1920x1080）
- **注意**: 如果文件不存在，程序会报错并提示

### font.ttf
- **说明**: 用于绘制文字的字体文件
- **格式**: TrueType 字体文件 (.ttf)
- **注意**: 如果文件不存在，程序会自动使用系统默认字体

## 使用说明

1. 将你的背景底图重命名为 `template.jpg` 并放置在此目录
2. （可选）将你喜欢的字体文件重命名为 `font.ttf` 并放置在此目录
3. 确保文件权限正确，程序可以读取这些文件
